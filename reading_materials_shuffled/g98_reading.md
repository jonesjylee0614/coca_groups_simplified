## Part 5. 语法聚焦 (Grammar Focus)

### 聚焦点 1: whoever 引导的名词性从句

**原句:** "Whoever donated them wanted to remain anonymous."

**聚焦:** whoever 引导主语从句

**用法说明:** 
whoever 是复合关系代词,可以引导名词性从句(主语从句、宾语从句等),表示"无论谁、任何...的人"。whoever 在从句中充当主语或宾语。

**与 who 的区别:**
- **who**: 引导疑问句或定语从句,表示特定的"谁"
- **whoever**: 表示"无论谁",强调任意性

**从句类型:**
1. 主语从句: Whoever comes is welcome. (无论谁来都欢迎)
2. 宾语从句: Give it to whoever needs it. (把它给任何需要的人)

**类似词:**
- whatever (无论什么)
- whichever (无论哪个)
- whomever (无论谁 - 宾格)

---

### 聚焦点 2: "create" 的常用搭配

**原句:** "Each display created a conversation about family life."

**聚焦:** create 与抽象名词的搭配

**用法说明:**
create 作及物动词,除了表示"创造"具体事物,还常与抽象名词搭配,表示"产生、引起"某种状态或氛围。这种用法在学术和正式写作中很常见。

**常用搭配:**
- create a conversation (引发对话)
- create an opportunity (创造机会)
- create a problem (制造问题)
- create an atmosphere (营造氛围)
- create awareness (提高意识)
- create confusion (造成混乱)

**例句:**
- The new policy created controversy. (新政策引发了争议)
- Music creates emotion. (音乐产生情感)
- This program creates jobs. (这个项目创造就业机会)
