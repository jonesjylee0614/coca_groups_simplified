# 🚀 如何启动网站 - 快速指南

## ⚠️ 重要提示

由于浏览器的安全限制（CORS策略），**不能直接双击 index.html 打开**，必须通过本地服务器运行。

## 📌 最简单的方法（Windows用户）

### 双击运行 `启动网站.bat`

1. 找到项目文件夹中的 **启动网站.bat** 文件
2. 双击运行
3. 等待浏览器自动打开
4. 开始学习！

**注意**: 需要安装 Python 环境。如果没有，请看下面的安装方法。

---

## 🐍 方法一：使用 Python（推荐）

### Windows用户

**方式1：使用批处理文件**
```bash
双击 "启动网站.bat"
```

**方式2：命令行**
```bash
# 打开命令提示符（cmd）或 PowerShell
# 进入项目目录，然后运行：
python start_server.py
```

### Mac/Linux用户

```bash
# 打开终端
# 进入项目目录，然后运行：
python3 start_server.py

# 或者直接使用 Python 内置服务器：
python3 -m http.server 8000
```

### 安装Python（如果还没有）

**Windows:**
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.x 最新版本
3. 安装时**务必勾选** "Add Python to PATH"
4. 安装完成后重新运行启动脚本

**Mac:**
```bash
# Mac 通常已预装 Python
# 如需更新，可使用 Homebrew:
brew install python3
```

**Linux:**
```bash
# Ubuntu/Debian
sudo apt-get install python3

# CentOS/RHEL
sudo yum install python3
```

## 📦 方法二：使用 Node.js

如果你已经安装了 Node.js：

```bash
# 使用 npx（无需安装）
npx http-server -p 8000

# 或安装后使用
npm install -g http-server
http-server -p 8000
```

安装 Node.js：访问 https://nodejs.org/

---

## 💻 方法三：使用 VSCode（推荐给开发者）

1. 安装 Visual Studio Code: https://code.visualstudio.com/
2. 打开项目文件夹
3. 安装插件 "Live Server" (由 Ritwick Dey 开发)
4. 右键点击 `index.html`
5. 选择 "Open with Live Server"

## 🔧 常见问题

### Q1: 提示"端口已被占用"
**原因**: 端口 8000 已被其他程序使用

**解决方法**:
- 关闭其他占用 8000 端口的程序
- 或修改 `start_server.py` 中的 `PORT = 8000` 为其他端口，如 `PORT = 8080`

### Q2: Python 未找到
**原因**: Python 未安装或未添加到环境变量

**解决方法**:
1. 重新安装 Python，记得勾选 "Add Python to PATH"
2. 或手动添加 Python 到系统环境变量

### Q3: 浏览器没有自动打开
**原因**: 自动打开浏览器功能失败（正常现象）

**解决方法**:
手动打开浏览器，访问控制台显示的地址（通常是 http://localhost:8000）

### Q4: 页面显示空白或报错
**原因**:
- 服务器未正确启动
- 端口被占用
- 浏览器缓存问题

**解决方法**:
1. 确认服务器正在运行
2. 检查控制台是否有错误信息
3. 按 Ctrl+F5 强制刷新页面
4. 尝试清除浏览器缓存

### Q5: 为什么不能直接打开 HTML 文件？
**原因**: 浏览器的 CORS（跨域资源共享）安全策略

现代浏览器不允许 `file://` 协议的页面使用 `fetch` API 加载本地文件，这是为了安全考虑。必须通过 HTTP(S) 协议访问。

## 🎯 推荐方式总结

| 用户类型 | 推荐方法 | 优点 |
|---------|---------|------|
| 普通用户 | 双击 `启动网站.bat` | 最简单，一键启动 |

| 开发者 | VSCode + Live Server | 方便开发调试 |
