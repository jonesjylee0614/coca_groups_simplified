# COCA 5000 高频词学习网站 - 使用说明

## 📚 项目简介

这是一个基于 COCA (Corpus of Contemporary American English) 语料库的英语高频词学习网站。网站包含5000个最常用的英语单词，分为100组，每组50个单词，通过阅读材料和配套学习资源帮助学习者系统掌握英语核心词汇。

## 🚀 快速开始

### 方式一：直接打开（推荐）
直接双击 `index.html` 文件即可在浏览器中打开网站。

### 方式二：使用本地服务器
如果遇到文件加载问题，建议使用本地服务器：

**使用 Python:**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

**使用 Node.js:**
```bash
npx http-server -p 8000
```

然后在浏览器中访问: `http://localhost:8000`

## 📖 网站功能

### 首页 (index.html)

#### 1. 统计概览
- **总组数**: 显示共100组学习材料
- **已完成**: 显示已标记为完成的组数
- **总词汇量**: 5000个核心词汇
- **完成度**: 学习进度百分比

#### 2. 快速跳转
在搜索框中输入组号（1-100），点击 GO 按钮或按回车键，即可直接跳转到对应的学习页面。

#### 3. 分组浏览
- 100个组被分为10个批次显示（Group 1-10, 11-20, ...）
- 每个批次显示完成进度
- 点击任意批次卡片进入该批次的第一组

### 学习页面 (viewer.html)

#### 左侧 - 英文阅读区
- 显示原版英文阅读材料
- 重点词汇以加粗显示并高亮背景
- 可调节字体大小（A- / A+）
- 可切换单词高亮显示（👁️ 按钮）

#### 右侧 - 学习辅助区（6个标签页）

**1. 中文翻译**
- 完整的中文对照翻译
- 帮助理解文章内容

**2. 词汇讲解**
- 50个重点词汇的详细讲解
- 包含音标、用法、例句
- 记忆技巧和词汇搭配

**3. 句子分析**
- 重点句子的语法结构分析
- 词汇搭配和学习要点

**4. 记忆技巧**
- 各类词汇的记忆方法
- 联想记忆技巧
- 句型记忆法

**5. 练习建议**
- 朗读练习
- 替换练习
- 造句练习
- 情景对话
- 写作练习

**6. 我的笔记**
- 自定义学习笔记
- 自动保存到浏览器本地
- 每组独立的笔记空间

#### 顶部工具栏
- **返回首页**: 回到首页
- **👁️ 高亮**: 切换单词高亮显示
- **A- / A+**: 调整阅读区字体大小
- **☐ 完成**: 标记当前组为已完成/未完成

#### 底部导航
- **← 上一组**: 前往上一组（快捷键：←）
- **当前进度**: 显示当前组号和总组数
- **下一组 →**: 前往下一组（快捷键：→）

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| ← | 上一组 |
| → | 下一组 |
| C | 切换完成状态 |
| H | 切换高亮显示 |

## 💾 数据存储

所有学习数据都保存在浏览器的 localStorage 中，包括：

- ✅ 已完成的组列表
- 📝 每组的学习笔记
- 🔤 字体大小设置
- 👁️ 高亮显示设置

**注意**: 数据仅保存在当前浏览器中，清除浏览器缓存会删除这些数据。

## 📝 学习建议

### 推荐学习流程：

1. **第一遍：通读理解**
   - 先阅读英文原文，尝试理解大意
   - 对照中文翻译，确认理解正确

2. **第二遍：词汇学习**
   - 查看词汇讲解，学习每个单词
   - 注意音标、用法和例句

3. **第三遍：深入分析**
   - 阅读句子分析，理解语法结构
   - 学习记忆技巧，加深印象

4. **第四遍：实践练习**
   - 按照练习建议进行各种练习
   - 在笔记区记录心得和问题

5. **复习巩固**
   - 定期回顾已完成的组
   - 利用笔记进行复习

### 学习节奏建议：
- **每天1-2组**: 约2-3个月完成
- **每天3-5组**: 约1个月完成
- **根据个人情况**: 灵活调整

## 🐛 常见问题

### Q1: 为什么内容显示不正常？
A: 确保使用现代浏览器（Chrome、Firefox、Edge、Safari），或使用本地服务器打开。

### Q2: 我的学习进度丢失了怎么办？
A: 进度保存在浏览器localStorage中，不要清除浏览器缓存。建议定期导出笔记。

### Q3: 可以在手机上使用吗？
A: 可以！网站支持响应式设计，在手机上也能正常使用。

### Q4: 如何打印学习材料？
A: 在学习页面使用浏览器的打印功能（Ctrl+P），网站已优化打印样式。
